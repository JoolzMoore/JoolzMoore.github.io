What is the difference between accessing data from arrays and object literals?
Arrays let you create and group together data in interesting ways. Arrays are not a new data type, but lists with data values. Each value has an index, which is the numbered poistion of that value in the array. The first value us at index ), the next value is at index 1 etc. An Array is a subclass, or sub-prototype, of an Object, and it can inherit features from an Object. In JavaScript, an object literal is a comma separated list of named values wrapped in curly braces. It is created and has properties and methods added to it.

What is a function?
It is a way to bundle code so it can be re-used. Functions allow us to run the same piece of code from multiple places in a programme without having to copy and paste the code repeatedly.
