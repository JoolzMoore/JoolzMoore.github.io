How to use:


( )

parentheses
1. Execute a function (e.g. functionName(), console.log()),
2. Express conditons (e.g. if () {} else if () { }, for () { }, while () { })or var result = doSomething()


[ ]

square brackets
1. Create an array e.g. var arr = []
2. Point to an index in an array e.g. arr[i])
3. Point to a key in an obeject e.g. dinner.menu["pasta"]


{ }

curly braces
1. enclose conditional statements (e.g. if (...) {i+a} else if (...) {i-a}),
2. Function definitions (e.g. function myFunc() {with definition here}),
3. Loops (e.g. for () { looping around items in here}),
4. Declaring objects (e.g. var myObject = {"valueOne": "valueTwo"})


" " ' '


single and double quotes
1.Single quotes and double quotes can be interchangeable to denote that the text/numbers inside the quotes inside are a string.
2. Declaring objects (e.g. var myObject = {"valueOne": "valueTwo"})
